# Accounting App

Local-first accounting workspace with two browser apps:

- `Ledger`: monthly income, expenses, paid status, custom columns, JSON import/export.
- `Debts`: debt balances, monthly commitments, repayment progress, and payment logs.

The repository is scrubbed of personal seed data, spreadsheets, local TLS keys, and generated bytecode. Data is stored in the browser with `localStorage`; exports are intentionally ignored by git.

## Run

```bash
python3 server.py
```

Open `http://localhost:5173/Accounting/`.

The legacy app-specific servers still work:

```bash
python3 Ledger/server.py
python3 Debts/server.py
```

## Data

Seed files are empty by default:

- `Ledger/seed.js`
- `Debts/debts-seed.js`

Use the in-app import/export controls for private data. Do not commit exported JSON, spreadsheets, local certificates, or browser storage dumps.
